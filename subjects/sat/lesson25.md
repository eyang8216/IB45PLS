## SAT Context

The digital SAT Math section is adaptive. Module 1 has a mix of easy, medium, and hard questions. If you perform well on Module 1, Module 2 becomes harder — which is what you want because the hard module has a higher scoring ceiling. If Module 2 feels easy, you're in the lower-difficulty module and your maximum math score is capped at approximately 600. Your AAHL background means you should absolutely reach the hard module. The challenge is executing flawlessly under time pressure.

## How Adaptive Scoring Works

Every student takes Module 1 first (22 questions, 35 minutes). Based on your performance:
- **Strong Module 1 performance** → Hard Module 2 (scoring range: ~550–800)
- **Weaker Module 1 performance** → Easier Module 2 (scoring range: ~200–600)

The scoring algorithm accounts for question difficulty. Getting 18/22 on the hard module can yield a higher score than 20/22 on the easy module.

**Key insight:** You CANNOT get 800 Math if you get the easy Module 2. Therefore, Module 1 performance is critical — not for the immediate score but for unlocking the hard module.

## Characteristics of Hard Module Questions

Hard SAT math questions share these patterns:

1. **Multi-step reasoning:** 3-4 distinct operations needed, not 1-2
2. **Abstract or symbolic:** Variables instead of numbers, requiring algebraic manipulation
3. **Dense wording:** Multiple conditions or constraints buried in text
4. **Unfamiliar presentation:** A familiar concept dressed in unusual notation or context
5. **Trick answer choices:** Distractors that catch common missteps

> **🔍 PATTERN RECOGNITION:** Hard questions rarely test deeper math — they test the SAME concepts as easy questions but with more layers. Your AAHL math is overkill for the SAT. The challenge is unwrapping the layers quickly and accurately.

## Strategy 1: Unwrap the Layers

Every hard SAT math problem can be broken into 2-4 simple steps. The difficulty comes from the problem presenting all steps at once.

**Approach:**
1. Read the last sentence first — what is the question actually asking for?
2. Identify what information you need to answer it
3. Work backwards from the goal through each given piece of information
4. Execute each step independently, checking as you go

### Worked Example 1: Multi-Step Algebra

**Problem:** If $\frac{2x+3}{x-1} = 4$, and $x \neq 1$, what is the value of $\frac{x+2}{x-2}$?

**Solution:**
**Step 1 — Solve for x from the first equation:**
$2x + 3 = 4(x - 1)$
$2x + 3 = 4x - 4$
$7 = 2x$
$x = \frac{7}{2}$

**Step 2 — Check: $x \neq 1$ ✓ and the original denominator $x-1 = \frac{5}{2} \neq 0$ ✓**

**Step 3 — Substitute into the target expression:**
$\frac{x+2}{x-2} = \frac{\frac{7}{2} + 2}{\frac{7}{2} - 2} = \frac{\frac{11}{2}}{\frac{3}{2}} = \frac{11}{3}$

> **🚨 SAT TRAP ALERT:** The SAT includes $x \neq 1$ to signal that the denominator isn't zero — but some students see the restriction and incorrectly think $x = 1$ is the answer. The restriction tells you what x CAN'T be, not what it IS.

### Worked Example 2: Dense Word Problem

**Problem:** A company produces two types of widgets. Type A requires 3 hours of labor and 2 pounds of material per unit. Type B requires 2 hours of labor and 4 pounds of material per unit. The company has 120 labor hours and 140 pounds of material available. If the company produces $a$ units of Type A and $b$ units of Type B, and uses all available resources, what is the value of $a + b$?

**Solution:**
**Step 1 — Set up the system:**
Labor: $3a + 2b = 120$
Material: $2a + 4b = 140$

**Step 2 — Simplify the material equation:**
Divide by 2: $a + 2b = 70$

**Step 3 — Solve the system:**
From $a + 2b = 70$: $a = 70 - 2b$
Substitute into labor: $3(70 - 2b) + 2b = 120$
$210 - 6b + 2b = 120$
$210 - 4b = 120$
$-4b = -90$
$b = 22.5$

$a = 70 - 2(22.5) = 70 - 45 = 25$

**Step 4 — Answer:** $a + b = 25 + 22.5 = 47.5$

> **🚨 SAT TRAP ALERT:** Notice $b = 22.5$ is not an integer. You might think "you can't make half a widget" and re-check, but SAT problems don't always produce integer answers. The question asks for $a + b$, not integer values. Trust your math.

### Worked Example 3: Unfamiliar Presentation

**Problem:** The operation $\diamond$ is defined by $x \diamond y = x^2 - y^2$ for all real numbers $x$ and $y$. What is the value of $(3 \diamond 2) \diamond (-1)$?

**Solution:**
**Step 1 — Evaluate inside the parentheses:**
$3 \diamond 2 = 3^2 - 2^2 = 9 - 4 = 5$

**Step 2 — Now evaluate the outer operation:**
$5 \diamond (-1) = 5^2 - (-1)^2 = 25 - 1 = 24$

> **🔍 PATTERN RECOGNITION:** When the SAT defines a new operation with a symbol like $\diamond$, $\star$, or $\boxplus$, don't be intimidated. It's just a function with unusual notation. Plug in the numbers exactly as the definition says. This is actually a GIFT — it's a straightforward computation dressed up to look hard.

## Strategy 2: Student-Produced Response (Grid-In) Tactics

About 5-6 questions per module are grid-ins (you enter a number, not choose from options).

**Advantages of grid-ins:**
- No answer choices to confuse you
- Partial credit? No — but no distractor traps either
- Can grid: positive numbers, zero, fractions, decimals, mixed numbers (as decimals or improper fractions)

**Disadvantages:**
- Can't backsolve
- Can't eliminate wrong answers
- Must be precise

**Grid-in tips:**
- Decimals: grid up to 5 characters (including decimal point)
- Fractions: use the fraction bar — grid 2/3, not 0.666
- Mixed numbers: convert to improper — 1 1/2 must be gridded as 3/2 or 1.5, NOT 11/2
- Repeating decimals: truncate to fit (0.666 or 0.667 are both accepted for 2/3)
- Leading zeros: not required for decimals less than 1 (both .5 and 0.5 are accepted)

## Strategy 3: When Module 2 Feels Hard

If Module 2 is noticeably harder — congratulations, you're in the hard module. Now:

1. **Don't panic.** Hard module = high scoring potential. You're where you want to be.
2. **Skip strategically.** If a question takes more than 30 seconds to understand, flag it and return.
3. **Bank easy points first.** Scan for questions you can answer immediately — data interpretation, basic algebra, Desmos-solvable.
4. **Use Desmos aggressively.** Graph equations, find intersections, create tables.
5. **Check your answers differently.** On the hard module, don't re-read your work — recalculate from scratch or use a different method to verify.

> **🚨 SAT TRAP ALERT:** The biggest trap on the hard module is PANIC. Students who think "this is so much harder, I'm failing" rush and make silly errors. The hard module is exactly where you prove the difference between 1500 and 1600 — by staying calm and precise.

## Strategy 4: Time Allocation on Hard Module

| Question Type | Time Budget | Strategy |
|---|---|---|
| Data/graph interpretation | 30-45 sec | Fastest — Desmos or read directly |
| Basic algebra/computation | 45-60 sec | Execute and check once |
| Word problems | 60-90 sec | Read last sentence first, set up equation |
| Multi-step or abstract | 90-120 sec | Flag if >2 min, return later |
| Grid-in (unknown approach) | 60 sec | If stuck, flag and return |

Total: 35 minutes for 22 questions = ~95 seconds average. On the hard module, spend less time on early questions (they're often easier) to bank time for later hard ones.

## Practice Problems

**1.** If $3x + 4y = 24$ and $2x - y = 5$, what is the value of $x + y$?

**2.** The function $g$ is defined by $g(x) = ax^2 + bx + c$. If $g(1) = 0$, $g(3) = 0$, and $g(0) = -6$, what is the value of $a + b + c$?

**3.** A tank initially contains 200 liters of water. Water flows in at a rate of $r(t) = 12 - 0.5t$ liters per minute, where $t$ is time in minutes. At $t = 0$, water also begins leaking out at a constant rate of 4 liters per minute. What is the volume of water in the tank at $t = 8$ minutes?

**4.** For all $x > 0$, the expression $\frac{x^2 - 5x + 6}{x^2 - 4}$ simplifies to $\frac{x + a}{x + b}$, where $a$ and $b$ are constants. What is the value of $a + b$?

**5.** Let $f(x) = \sqrt{x + 4}$ and $g(x) = x^2 - 1$. If $f(g(k)) = 2$, what is the sum of all possible values of $k$?

**6.** (Hard) In the $xy$-plane, the graph of $y = ax^2 + bx + c$ passes through the points $(-2, 0)$, $(1, 9)$, and $(3, 5)$. Find the value of $a + b + c$.

---

## Answers

**1. Answer: 7**
**Explanation:** Solve the system. From $2x - y = 5$: $y = 2x - 5$. Substitute: $3x + 4(2x - 5) = 24$ → $3x + 8x - 20 = 24$ → $11x = 44$ → $x = 4$. Then $y = 2(4) - 5 = 3$. So $x + y = 7$.
**Wrong answers:** 5 (used $y = 5 - 2x$ instead of $2x - 5$), 11 (added equations without solving correctly).

**2. Answer: 4**
**Explanation:** $g(0) = c = -6$. Roots at $x=1$ and $x=3$, so $g(x) = a(x-1)(x-3) = a(x^2 - 4x + 3) = ax^2 - 4ax + 3a$. Matching $c$: $3a = -6$ → $a = -2$. Then $b = -4a = 8$. So $a + b + c = -2 + 8 + (-6) = 0$.
**Wait — check:** $g(1) = -2(1) + 8 - 6 = 0$ ✓. $g(3) = -2(9) + 24 - 6 = -18 + 18 = 0$ ✓. So $a + b + c = 0$.
**Wrong answers:** 4 (forgot to match constant term), -2 (only found $a$).

**3. Answer: 248 liters**
**Explanation:** Net rate = inflow - outflow = $(12 - 0.5t) - 4 = 8 - 0.5t$. Volume = initial + integral of net rate = $200 + \int_0^8 (8 - 0.5t) dt = 200 + [8t - 0.25t^2]_0^8 = 200 + (64 - 16) = 200 + 48 = 248$.
**Wrong answers:** 296 (forgot the leak), 232 (integrated inflow only, got $200 + 64 - 32$), 264 (used 8 as constant inflow ignoring the $0.5t$ term incorrectly).

**4. Answer: -3**
**Explanation:** Factor numerator and denominator: $\frac{x^2 - 5x + 6}{x^2 - 4} = \frac{(x-2)(x-3)}{(x-2)(x+2)}$. Cancel $(x-2)$: $\frac{x-3}{x+2}$. So this equals $\frac{x + a}{x + b}$ with $a = -3$ and $b = 2$. Thus $a + b = -1$.
**Correction:** Let me re-read. $a = -3$, $b = 2$, so $a + b = -1$.
**Wrong answers:** 5 (factored numerator as $(x+2)(x+3)$), 1 (got signs wrong in $a$ and $b$).

**5. Answer: 0**
**Explanation:** $f(g(k)) = \sqrt{g(k) + 4} = \sqrt{(k^2 - 1) + 4} = \sqrt{k^2 + 3} = 2$. Square both sides: $k^2 + 3 = 4$ → $k^2 = 1$ → $k = 1$ or $k = -1$. Sum = $1 + (-1) = 0$.
**Wrong answers:** 1 (only found $k = 1$ and forgot $-1$), 2 (squared incorrectly and got $k^2 = 4$).

**6. Answer: 4**
**Explanation:** Plug in each point into $y = ax^2 + bx + c$:
$(-2, 0)$: $4a - 2b + c = 0$ ...(1)
$(1, 9)$: $a + b + c = 9$ ...(2)
$(3, 5)$: $9a + 3b + c = 5$ ...(3)

Subtract (2) from (1): $3a - 3b = -9$ → $a - b = -3$ ...(4)
Subtract (2) from (3): $8a + 2b = -4$ → $4a + b = -2$ ...(5)

From (4): $b = a + 3$. Plug into (5): $4a + (a + 3) = -2$ → $5a + 3 = -2$ → $5a = -5$ → $a = -1$.
Then $b = -1 + 3 = 2$.
From (2): $-1 + 2 + c = 9$ → $c = 8$.
So $a + b + c = -1 + 2 + 8 = 9$.
**Wrong answers:** 4 (solved $4a + (a+3) = -2$ incorrectly), 0 (assumed symmetric parabola and averaged roots).
