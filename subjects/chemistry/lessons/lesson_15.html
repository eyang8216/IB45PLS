Lesson 15: Hess's Law — Enthalpy Cycles, ΔHf⦵ & ΔHc⦵ Methods

Syllabus reference: 5.2 — Hess's Law

Learning Objectives

By the end of this lesson, you will be able to:

  • State Hess's Law and explain why it is a consequence of enthalpy as a state function.
  • Construct enthalpy cycles using $\Delta H_f^\ominus$ data (formation method) to calculate an unknown $\Delta H$.
  • Construct enthalpy cycles using $\Delta H_c^\ominus$ data (combustion method) to calculate an unknown $\Delta H$.
  • Use algebraic addition of thermochemical equations to determine enthalpy changes indirectly.

1. Prior Knowledge Check

Question 1: What does it mean that enthalpy is a "state function"?

▸ Show
A state function is a property whose value depends only on the current state of the system (temperature, pressure, composition), not on the path taken to reach that state. Enthalpy ($H$) is a state function — the enthalpy change between reactants and products is the same whether the reaction happens in one step or through multiple intermediate steps. This is the entire basis of Hess's Law.

Question 2: Write a balanced equation for the formation of $\ce{H2SO4(l)}$ from its elements in their standard states.

▸ Show
$\ce{H2(g) + S(s) + 2O2(g) -> H2SO4(l)}$. The elements must be in their standard states at 298 K and 100 kPa: $\ce{H2}$ is a gas, $\ce{S}$ is solid (rhombic), $\ce{O2}$ is a gas. Note that this produces exactly one mole of $\ce{H2SO4}$.

Question 3: What is the sign of $\Delta H_c^\ominus$ for all substances, and why?

▸ Show
$\Delta H_c^\ominus$ is always negative (exothermic) because combustion releases energy. By definition, combustion is the reaction of a substance with oxygen to form the most stable oxides — and breaking bonds in the fuel plus forming strong $\ce{C=O}$ and $\ce{O-H}$ bonds in $\ce{CO2}$ and $\ce{H2O}$ always releases net energy.

2. Hess's Law — The Statement

Hess's Law: The enthalpy change for a reaction is the same regardless of the route taken, provided the initial and final conditions are the same. Mathematically: if a reaction can be expressed as the sum of two or more steps, then $\Delta H_{\text{total}} = \Delta H_1 + \Delta H_2 + \Delta H_3 + \cdots$

Why does this matter? Because you cannot always measure $\Delta H$ directly. For example, you cannot simply put carbon and hydrogen together and expect them to form ethane — the direct reaction is impractical. But you can burn carbon, hydrogen, and ethane separately, measure each $\Delta H_c$, and use Hess's Law to find $\Delta H_f$ of ethane.

direct Intermediate A Route 1: ΔH₁ Route 2: ΔH₂ ΔHdirect = ΔH₁ + ΔH₂ The path does not matter — only the endpoints.
Figure 1: Hess's Law illustrated. The direct enthalpy change equals the sum of the indirect route — because enthalpy is a state function and the start/end points are identical.

Check-in: Given: $\ce{C(s) + O2(g) -> CO2(g)}$, $\Delta H = -394\text{ kJ mol}^{-1}$ and $\ce{CO(g) + 1/2 O2(g) -> CO2(g)}$, $\Delta H = -283\text{ kJ mol}^{-1}$. Find $\Delta H$ for $\ce{C(s) + 1/2 O2(g) -> CO(g)}$.

▸ Show
Think of the route: C + O₂ → CO₂ (−394) is the direct combustion. The indirect route: C + ½O₂ → CO (unknown $x$), then CO + ½O₂ → CO₂ (−283). By Hess's Law: $x + (-283) = -394$, so $x = -394 + 283 = \mathbf{-111\text{ kJ mol}^{-1}}$.

3. The Two Standard Methods

3.1 Method 1: Using $\Delta H_f^\ominus$ (Formation Method)

This method works for any reaction where you have $\Delta H_f^\ominus$ data for all reactants and products. The general cycle goes from elements (enthalpy baseline) up to compounds:

$$\Delta H_{\text{reaction}}^\ominus = \sum \Delta H_f^\ominus(\text{products}) - \sum \Delta H_f^\ominus(\text{reactants})$$
f Method Elements in standard states Reactants (compounds) Products (compounds) ΔHrxn −ΣΔH°f(reactants) −ΣΔH°f(products)
Figure 2: The formation cycle. All compounds are "formed" from elements. Going clockwise: reactants → elements is $-\sum\Delta H_f^\ominus$(reactants); elements → products is $+\sum\Delta H_f^\ominus$(products). The sum equals the direct reaction arrow.

Check-in: Use $\Delta H_f^\ominus$ data to find $\Delta H^\ominus$ for: $\ce{Fe2O3(s) + 3CO(g) -> 2Fe(s) + 3CO2(g)}$. Data (kJ mol⁻¹): $\Delta H_f^\ominus(\ce{Fe2O3}) = -824$, $\Delta H_f^\ominus(\ce{CO}) = -111$, $\Delta H_f^\ominus(\ce{CO2}) = -394$.

▸ Show
$\Delta H^\ominus = \sum\Delta H_f^\ominus(\text{prod}) - \sum\Delta H_f^\ominus(\text{react})$. Products: $2(0) + 3(-394) = -1182$ (Fe element = 0). Reactants: $(-824) + 3(-111) = -824 - 333 = -1157$. $\Delta H^\ominus = -1182 - (-1157) = \mathbf{-25\text{ kJ mol}^{-1}}$.

3.2 Method 2: Using $\Delta H_c^\ominus$ (Combustion Method)

This method is used when the reaction involves organic compounds (which burn easily). The cycle goes from compounds down to combustion products ($\ce{CO2}$ and $\ce{H2O}$):

$$\Delta H_{\text{reaction}}^\ominus = \sum \Delta H_c^\ominus(\text{reactants}) - \sum \Delta H_c^\ominus(\text{products})$$

Note the order is swapped compared to the formation method! Reactants come first. This is because the cycle goes down to combustion products rather than up from elements.

c Method Reactants (organic compounds) Products (organic compounds) ΔHrxn CO₂(g) + H₂O(l) (combustion products) ΣΔH°c(reactants) ΣΔH°c(products)
Figure 3: The combustion cycle. All organic compounds are "burned down" to CO₂ and H₂O. Going clockwise: reactants → combustion products → products. Note that $\Delta H^\ominus_{\text{rxn}} = \sum\Delta H_c^\ominus(\text{reactants}) - \sum\Delta H_c^\ominus(\text{products})$.

Check-in: Given: $\Delta H_c^\ominus(\ce{C(s)}) = -394$, $\Delta H_c^\ominus(\ce{H2(g)}) = -286$, $\Delta H_c^\ominus(\ce{C2H6(g)}) = -1560$ kJ mol⁻¹. Find $\Delta H_f^\ominus$ of $\ce{C2H6(g)}$.

▸ Show
Formation reaction: $\ce{2C(s) + 3H2(g) -> C2H6(g)}$. Using combustion cycle: $\Delta H^\ominus = \sum\Delta H_c^\ominus(\text{reactants}) - \sum\Delta H_c^\ominus(\text{products}) = [2(-394) + 3(-286)] - [-1560] = [-788 - 858] + 1560 = -1646 + 1560 = \mathbf{-86\text{ kJ mol}^{-1}}$.

4. Worked Examples

Example 1 — Formation Method with Three Compounds:

Calculate $\Delta H^\ominus$ for: $\ce{2Al(s) + Fe2O3(s) -> Al2O3(s) + 2Fe(s)}$ (the thermite reaction). Data: $\Delta H_f^\ominus(\ce{Fe2O3}) = -824$, $\Delta H_f^\ominus(\ce{Al2O3}) = -1676$ kJ mol⁻¹.

Step 1: $\sum\Delta H_f^\ominus(\text{products}) = (-1676) + 2(0) = -1676$ kJ.
Step 2: $\sum\Delta H_f^\ominus(\text{reactants}) = 2(0) + (-824) = -824$ kJ.
Step 3: $\Delta H^\ominus = -1676 - (-824) = \mathbf{-852\text{ kJ mol}^{-1}}$.

This is highly exothermic — which is why thermite is used for welding railway tracks!

⚠️ Common wrong approach: Forgetting that elements (Al, Fe) have $\Delta H_f^\ominus = 0$. Students sometimes look them up or invent a value, which ruins the calculation.
Example 2 — Combustion Method for Isomerisation:

Calculate $\Delta H^\ominus$ for the isomerisation: $\ce{CH3CH2CH2OH(l) -> (CH3)2CHOH(l)}$. Data (kJ mol⁻¹): $\Delta H_c^\ominus(\text{propan-1-ol}) = -2021$, $\Delta H_c^\ominus(\text{propan-2-ol}) = -2005$.

Step 1: Identify which is reactant and which is product.
Step 2: $\Delta H^\ominus = \Delta H_c^\ominus(\text{reactant}) - \Delta H_c^\ominus(\text{product}) = (-2021) - (-2005) = -2021 + 2005 = \mathbf{-16\text{ kJ mol}^{-1}}$.

The reaction is slightly exothermic — propan-2-ol is marginally more stable.

⚠️ Common wrong approach: Reversing the order: products minus reactants. With the combustion method, the formula is reactants minus products. If you get the sign wrong here, you lose both marks.
Example 3 — Multi-step Algebraic Addition:

Given the following data, calculate $\Delta H^\ominus$ for $\ce{N2(g) + 2O2(g) -> 2NO2(g)}$:
(1) $\ce{N2(g) + O2(g) -> 2NO(g)}$, $\Delta H = +180\text{ kJ}$
(2) $\ce{2NO(g) + O2(g) -> 2NO2(g)}$, $\Delta H = -114\text{ kJ}$

Step 1: Add equations (1) and (2): $\ce{N2 + O2 + 2NO + O2 -> 2NO + 2NO2}$.
Step 2: Cancel $\ce{2NO}$ on both sides: $\ce{N2 + 2O2 -> 2NO2}$.
Step 3: $\Delta H = (+180) + (-114) = \mathbf{+66\text{ kJ}}$.

⚠️ Common wrong approach: Failing to check that the intermediate ($\ce{2NO}$) cancels exactly. If the coefficients don't match, you must multiply one equation first — Hess's Law allows multiplying $\Delta H$ by the same factor.

⚠️ Common Mistakes to Avoid

  1. "Using the formation formula for a combustion cycle, or vice versa." Formation: $\Delta H = \sum\Delta H_f(\text{products}) - \sum\Delta H_f(\text{reactants})$. Combustion: $\Delta H = \sum\Delta H_c(\text{reactants}) - \sum\Delta H_c(\text{products})$. The order is reversed. Always draw the cycle and trace arrows; never memorise the formula blindly.
  2. "Forgetting that $\Delta H_f^\ominus$ of elements = 0." This is the most common arithmetic error. Fe(s), O₂(g), Al(s), C(s, graphite) all have $\Delta H_f^\ominus = 0$. Including a non-zero value for an element will give a wrong answer that examiners will spot immediately.
  3. "Not multiplying $\Delta H$ by the stoichiometric coefficient." If the balanced equation has $3\ce{CO2}$, you need $3 \times \Delta H_f^\ominus(\ce{CO2})$. Every term in the sum must be multiplied by its coefficient from the balanced equation.
  4. "Confusing $\Delta H_c^\ominus$ of $\ce{H2O}$ — is it water liquid or gas?" Combustion produces $\ce{H2O(l)}$ at 298 K, so $\Delta H_c^\ominus$ data tables assume liquid water. If the question gives $\ce{H2O(g)}$, the value will be different (by the enthalpy of vaporisation, ~+44 kJ mol⁻¹). Read the state symbols.

5. End-of-Lesson Practice

Question 1 (Paper 1 style): Given: $\Delta H_f^\ominus(\ce{C2H5OH(l)}) = -278$ kJ mol⁻¹. Which equation does this value refer to?
A. $\ce{C2H5OH(l) + 3O2(g) -> 2CO2(g) + 3H2O(l)}$
B. $\ce{2C(s) + 3H2(g) + 1/2 O2(g) -> C2H5OH(l)}$
C. $\ce{C2H4(g) + H2O(l) -> C2H5OH(l)}$
D. $\ce{4C(s) + 6H2(g) + O2(g) -> 2C2H5OH(l)}$

Question 2: Calculate $\Delta H^\ominus$ for $\ce{CH4(g) + 2O2(g) -> CO2(g) + 2H2O(l)}$ using $\Delta H_f^\ominus$ values (kJ mol⁻¹): $\ce{CH4(g)} = -75$, $\ce{CO2(g)} = -394$, $\ce{H2O(l)} = -286$.

Question 3: Use combustion data to find $\Delta H^\ominus$ for $\ce{C2H4(g) + H2(g) -> C2H6(g)}$. Data (kJ mol⁻¹): $\Delta H_c^\ominus(\ce{C2H4}) = -1411$, $\Delta H_c^\ominus(\ce{H2}) = -286$, $\Delta H_c^\ominus(\ce{C2H6}) = -1560$.

Question 4: Given: (1) $\ce{C(s) + O2(g) -> CO2(g)}$, $\Delta H = -394$ kJ; (2) $\ce{H2(g) + 1/2 O2(g) -> H2O(l)}$, $\Delta H = -286$ kJ; (3) $\ce{C3H8(g) + 5O2(g) -> 3CO2(g) + 4H2O(l)}$, $\Delta H = -2220$ kJ. Calculate $\Delta H_f^\ominus$ of $\ce{C3H8(g)}$.

Question 5: Calculate $\Delta H^\ominus$ for $\ce{4NH3(g) + 5O2(g) -> 4NO(g) + 6H2O(g)}$ using $\Delta H_f^\ominus$ (kJ mol⁻¹): $\ce{NH3(g)} = -46$, $\ce{NO(g)} = +90$, $\ce{H2O(g)} = -242$.

Question 6 (IB-style): A student wants to determine $\Delta H^\ominus$ for the decomposition of $\ce{CaCO3}$. She has the following data (kJ mol⁻¹): $\Delta H_f^\ominus(\ce{CaCO3}) = -1207$, $\Delta H_f^\ominus(\ce{CaO}) = -635$, $\Delta H_f^\ominus(\ce{CO2}) = -394$.
(a) Write the balanced equation for the decomposition of $\ce{CaCO3}$.
(b) Calculate $\Delta H^\ominus$ for this reaction.
(c) Is the reaction exothermic or endothermic? Justify.
(d) Hence explain why $\ce{CaCO3}$ must be heated strongly for decomposition to occur.

Answer 1: B. The formation reaction starts from elements in their standard states and produces exactly one mole of the compound. A is combustion, C is hydration, D produces two moles (coefficient 2 is not allowed for $\Delta H_f$).

Answer 2: $\sum\Delta H_f^\ominus(\text{products}) = (-394) + 2(-286) = -394 - 572 = -966$. $\sum\Delta H_f^\ominus(\text{reactants}) = (-75) + 2(0) = -75$. $\Delta H^\ominus = -966 - (-75) = \mathbf{-891\text{ kJ mol}^{-1}}$. This is the standard enthalpy of combustion of methane.

Answer 3: $\Delta H^\ominus = \sum\Delta H_c^\ominus(\text{reactants}) - \sum\Delta H_c^\ominus(\text{products}) = [(-1411) + (-286)] - [-1560] = -1697 + 1560 = \mathbf{-137\text{ kJ mol}^{-1}}$.

Answer 4: Formation: $\ce{3C(s) + 4H2(g) -> C3H8(g)}$. Construct $\ce{C3H8}$ from elements. Hess cycle: $\Delta H_f = 3(-394) + 4(-286) - (-2220) = -1182 - 1144 + 2220 = \mathbf{-106\text{ kJ mol}^{-1}}$.

Answer 5: Products: $4(+90) + 6(-242) = 360 - 1452 = -1092$. Reactants: $4(-46) + 5(0) = -184$. $\Delta H^\ominus = -1092 - (-184) = \mathbf{-908\text{ kJ mol}^{-1}}$.

Answer 6: (a) $\ce{CaCO3(s) -> CaO(s) + CO2(g)}$. (b) $\Delta H^\ominus = [(-635) + (-394)] - [-1207] = -1029 + 1207 = \mathbf{+178\text{ kJ mol}^{-1}}$. (c) Endothermic — $\Delta H > 0$. (d) The reaction requires energy input to break bonds in the carbonate ion. Heating provides the activation energy and the endothermic enthalpy change; at room temperature, the reverse reaction (quicklime absorbing $\ce{CO2}$) is spontaneous.

6. Key Terms

TermDefinition
Hess's LawThe enthalpy change for a reaction is independent of the route taken; $\Delta H$ is the same whether the reaction occurs in one step or multiple steps
State functionA property that depends only on the current state of the system, not the path taken to reach it (e.g., enthalpy, internal energy, entropy)
Enthalpy cycleA diagram showing alternative routes between reactants and products, used to calculate an unknown $\Delta H$ using known enthalpy changes
$\Delta H_f^\ominus$ methodCycle through elements in standard states: $\Delta H_{\text{rxn}} = \sum\Delta H_f^\ominus(\text{products}) - \sum\Delta H_f^\ominus(\text{reactants})$
$\Delta H_c^\ominus$ methodCycle through combustion products: $\Delta H_{\text{rxn}} = \sum\Delta H_c^\ominus(\text{reactants}) - \sum\Delta H_c^\ominus(\text{products})$
Standard state of an elementThe most stable form of the element at 298 K and 100 kPa; $\Delta H_f^\ominus = 0$ by definition
Algebraic addition methodCombining thermochemical equations (with their $\Delta H$ values) to obtain a target equation; intermediates must cancel

7. Exam Tips

Paper 1 (Multiple Choice): Be ready for questions that give you 3–4 equations with $\Delta H$ values and ask you to find the $\Delta H$ for a target reaction by combining them. The method: identify the target equation, then decide which given equations to reverse (flip the sign of $\Delta H$) and which to multiply (multiply $\Delta H$ by the same factor). The intermediates must cancel exactly.
Paper 2 — Hess Cycle Construction: Always draw the cycle on the exam paper — even if the question doesn't explicitly ask for it. The cycle earns method marks, and if you make an arithmetic error, the examiner can still see you understood the principle. Label each arrow with the compound name and its $\Delta H$ value. Tip: For formation cycles, put elements at the bottom. For combustion cycles, put $\ce{CO2 + H2O}$ at the bottom.
Paper 2 — Data Booklet: The IB Data Booklet (Section 12) contains $\Delta H_f^\ominus$ and $\Delta H_c^\ominus$ values. You are expected to select the correct ones for your calculation. Check state symbols — $\ce{H2O(l)}$ vs $\ce{H2O(g)}$ have different values, and using the wrong one is a common error in exams.
