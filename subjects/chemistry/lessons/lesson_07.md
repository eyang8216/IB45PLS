### Problem 1: Subatomic Particles and Isotopes (6 marks)

The element gallium has the atomic number Z = 31. Naturally occurring gallium consists of two stable isotopes: gallium-69 and gallium-71. The relative abundance of gallium-69 is 60.1% and the relative abundance of gallium-71 is 39.9%.

(a) State the number of protons, the number of neutrons, and the number of electrons in a neutral atom of gallium-71. Write your answers clearly, one on each line. (3 marks)

(b) Calculate the relative atomic mass (Aᵣ) of gallium to one decimal place. Show your working. (2 marks)

(c) A sample of gallium is ionised in a mass spectrometer to produce Ga⁺ ions. State which ion — the \(\ce{^{69}Ga^+}\) ion or the \(\ce{^{71}Ga^+}\) ion — would be deflected more strongly by the magnetic field in the spectrometer, and explain your reasoning. (1 mark)

## Detailed Answers — IB Mark-Scheme Style

### Problem 1 — Solution

**(a) (3 marks)**

- Protons = 31. The atomic number Z = 31 tells us the number of protons. ✓
- Neutrons = 71 − 31 = 40. The mass number A = 71 is the total of protons and neutrons, so neutrons = A − Z. ✓
- Electrons = 31. In a neutral atom, the number of electrons equals the number of protons to balance the charge. ✓

**(b) (2 marks)**

The relative atomic mass is the weighted average of the masses of the naturally occurring isotopes. Using the formula:

\[
A_r = \frac{(69 \times 60.1) + (71 \times 39.9)}{100}
\]

\[
A_r = \frac{4146.9 + 2832.9}{100} = \frac{6979.8}{100} = 69.8
\]

One mark for the correct method (setting up the weighted average) and one mark for the correct answer to one decimal place. ✓✓

**(c) (1 mark)**

The \(\ce{^{69}Ga^+}\) ion would be deflected more strongly. ✓ The reason is that both ions carry the same charge (+1), but the \(\ce{^{69}Ga^+}\) ion has a smaller mass. In the magnetic field of a mass spectrometer, ions with a smaller mass-to-charge ratio (m/z) are deflected more because, for a given kinetic energy, lighter ions have a smaller radius of curvature and therefore bend more sharply.

### Problem 2 — Solution

**(a) (2 marks)**

Cobalt has Z = 27, so it has 27 electrons. Filling in the Aufbau order:

1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d⁷

One mark for the correct sequence of sub-levels and one mark for the correct number of electrons in each. ✓✓

Note: Writing 3d before 4s (1s² 2s² 2p⁶ 3s² 3p⁶ 3d⁷ 4s²) is also acceptable, and some periodic tables list configurations this way. However, the IBO typically accepts either order as long as the electron counts are correct.

**(b) (1 mark)**

To form Co²⁺, two electrons are removed. Electrons are removed from 4s before 3d. Therefore, Co²⁺ has the configuration:

[Ar] 3d⁷ ✓

**(c) (3 marks)**

The expected configuration for chromium (Z = 24), following the Aufbau order strictly, would be [Ar] 4s² 3d⁴. ✓

The actual configuration of chromium is [Ar] 4s¹ 3d⁵. ✓

The actual configuration is preferred because it gives a half-filled 3d sub-level (d⁵), in which each of the five 3d orbitals contains one electron with parallel spin. This half-filled arrangement has extra stability due to maximum exchange energy — a quantum mechanical stabilisation that occurs when electrons with parallel spins occupy different orbitals in the same sub-level. The energy gained by achieving this half-filled configuration outweighs the energy cost of promoting one electron from the 4s orbital to the 3d set. ✓

**(d) (2 marks)**

Copper (Z = 29) has the actual configuration [Ar] 4s¹ 3d¹⁰ (the copper exception: a fully-filled 3d sub-level is especially stable).

To form Cu²⁺, we remove two electrons: first the 4s¹ electron, then one electron from the 3d sub-level. This gives Cu²⁺: [Ar] 3d⁹.

The orbital diagram for the 3d⁹ sub-level is:

```
3d: [↑↓] [↑↓] [↑↓] [↑↓] [↑ ]
```

There is **1 unpaired electron**. ✓✓ (One mark for the correct diagram or description showing the arrangement, and one mark for the correct count of unpaired electrons.)

### Problem 3 — Solution

**(a) (2 marks)**

The line at 486 nm is in the visible region of the electromagnetic spectrum, so it belongs to the **Balmer series**. ✓

For the Balmer series, the electron always falls to n₁ = 2. This specific line at 486 nm is the second line of the series, corresponding to the transition from n₂ = 4 to n₁ = 2. ✓

**(b) (3 marks)**

First, convert the wavelength from nanometres to metres:

λ = 486 nm = 4.86 × 10⁻⁷ m ✓

Next, calculate the frequency using c = λν rearranged to ν = c/λ:

ν = (3.00 × 10⁸ m s⁻¹) / (4.86 × 10⁻⁷ m) = 6.17 × 10¹⁴ Hz ✓

Finally, calculate the photon energy using E = hν:

E = (6.63 × 10⁻³⁴ J s) × (6.17 × 10¹⁴ Hz) = 4.09 × 10⁻¹⁹ J ✓

(One mark for the wavelength conversion, one mark for the frequency calculation, one mark for the correct final energy.)

**(c) (3 marks)**

The **convergence limit** is the point in a spectral series at which the discrete lines become so close together that they merge into a continuous band of radiation. ✓ Physically, the convergence limit corresponds to a transition in which the electron falls from n = ∞ (completely free, no longer bound to the atom) to the lower level n₁ of the series. For the Lyman series (n₁ = 1), the convergence limit therefore represents the capture of a free electron by the nucleus into the n = 1 level. The reverse of this process — removing an electron from n = 1 to n = ∞ — is exactly the first ionisation of hydrogen.

To calculate the ionisation energy: Energy per atom = hν = (6.63 × 10⁻³⁴ J s) × (3.28 × 10¹⁵ Hz) = 2.17 × 10⁻¹⁸ J. ✓

Energy per mole = (2.17 × 10⁻¹⁸ J) × (6.02 × 10²³ mol⁻¹) = 1.31 × 10⁶ J mol⁻¹ = 1310 kJ mol⁻¹. ✓

(Award one mark for the energy per atom, one mark for the conversion to kJ mol⁻¹.)

### Problem 4 — Solution

**(a) (3 marks)**

The general trend is that first ionisation energy **increases** across Period 3 from sodium (496 kJ mol⁻¹) to argon (1521 kJ mol⁻¹). ✓

This increase occurs because the nuclear charge (the number of protons) increases from +11 in sodium to +18 in argon. ✓ The electrons being added across the period all enter the same shell (n = 3), and electrons in the same shell are poor at shielding one another. The shielding from the inner shells (1s² 2s² 2p⁶) remains constant at 10 inner electrons for every element across the period. ✓ Therefore, the effective nuclear charge experienced by the outermost electrons steadily increases while the distance from the nucleus and the inner-shell shielding stay approximately the same. The outer electrons are held more tightly with each step, requiring more energy to remove.

**(b) (2 marks)**

Magnesium has the electron configuration [Ne] 3s², so the electron being removed in the first ionisation is a 3s electron. Aluminium has the configuration [Ne] 3s² 3p¹, so the electron being removed is a 3p electron. ✓ The 3p orbital is higher in energy than the 3s orbital, and the 3p electron is, on average, slightly further from the nucleus. Even though aluminium has one more proton than magnesium (Z = 13 vs Z = 12), the fact that the electron is removed from the higher-energy 3p sub-level outweighs the increased nuclear attraction. Therefore, aluminium has a lower first ionisation energy. ✓

**(c) (2 marks)**

Phosphorus has the electron configuration [Ne] 3s² 3p³, with a half-filled 3p sub-level in which each of the three p orbitals contains exactly one electron with parallel spin. This half-filled arrangement has extra stability. Sulfur has the configuration [Ne] 3s² 3p⁴, in which one of the 3p orbitals must contain a pair of electrons. ✓ The repulsion between these two paired electrons occupying the same orbital makes one of them easier to remove than an unpaired electron from phosphorus's half-filled sub-level. This electron-electron repulsion outweighs the small increase in nuclear charge from phosphorus (Z = 15) to sulfur (Z = 16), resulting in a slightly lower first ionisation energy for sulfur. ✓

**(d) (3 marks)**

Argon has a lower first ionisation energy than neon even though argon has significantly more protons (18 vs 10). ✓ The reason is that argon's outermost electrons are in the n = 3 shell, whereas neon's are in the n = 2 shell — one shell further out from the nucleus. ✓ Because the electrostatic force decreases with the square of the distance, electrons further from the nucleus are held less tightly. Additionally, argon has more inner-shell electrons (the 1s² 2s² 2p⁶ electrons, totalling 10) shielding its outer electrons from the full nuclear charge, whereas neon has only the 1s² electrons (2 inner electrons) providing shielding. ✓ The increased distance and increased shielding together outweigh the increased nuclear charge, making argon's outer electrons easier to remove than neon's.

### Problem 5 — Solution

**(a) (2 marks)**

The second ionisation energy is the minimum energy required to remove one mole of electrons from one mole of singly charged gaseous cations, forming one mole of doubly charged gaseous cations. ✓

The equation for the second ionisation energy of element X is:

\[
\ce{X^+(g) -> X^{2+}(g) + e^-}
\]

✓ (One mark for the definition, one mark for the correct equation with state symbols.)

**(b) (2 marks)**

To identify the group, we look for the largest jump between successive ionisation energies. The ratios are:

- IE₂/IE₁ = 1577/786 ≈ 2.0
- IE₃/IE₂ = 3232/1577 ≈ 2.0
- IE₄/IE₃ = 4356/3232 ≈ 1.3
- IE₅/IE₄ = 16091/4356 ≈ 3.7 ← Largest jump
- IE₆/IE₅ = 19805/16091 ≈ 1.2

The largest jump occurs between IE₄ and IE₅. ✓ This indicates that the first four electrons are valence electrons (relatively easy to remove, all from the same outer shell) and the fifth electron comes from an inner shell (much more tightly held). Therefore, element X has 4 valence electrons and belongs to **Group 14**. ✓

**(c) (1 mark)**

Element X is in Period 3, Group 14. This is **silicon (Si)**. ✓

**(d) (2 marks)**

The first four ionisation energies (IE₁ through IE₄) remove the four valence electrons of silicon, which are all in the n = 3 shell (the 3s² and 3p² electrons). Because these electrons all come from the same principal shell, the energy increases among these first four IEs are relatively moderate. ✓

IE₅, however, must remove an electron from the n = 2 shell — specifically from the 2p⁶ sub-level that is part of the neon-like inner core ([Ne]). Electrons in the n = 2 shell are much closer to the nucleus than electrons in the n = 3 shell. They also experience far less shielding because there are fewer electrons between them and the nucleus. As a result, these inner-shell electrons are held enormously more tightly, and removing one requires dramatically more energy. This is why the increase from IE₄ (4356 kJ mol⁻¹) to IE₅ (16091 kJ mol⁻¹) is so large. ✓

**(e) (1 mark)**

The element immediately to the left of silicon in Period 3 is aluminium (Al), which has IE₁ = 578 kJ mol⁻¹. Aluminium has the electron configuration [Ne] 3s² 3p¹, so the electron removed in the first ionisation is a 3p electron. Silicon has the configuration [Ne] 3s² 3p² and also loses a 3p electron in its first ionisation, but silicon has one more proton (Z = 14 vs Z = 13). ✓ With both electrons coming from the 3p sub-level and the same inner-shell shielding, silicon's greater nuclear charge means its outer electron is held more tightly, resulting in a higher first ionisation energy than aluminium.

## Summary of Key Concepts Across the Atomic Structure Topic

The table below brings together the most important ideas from all six lessons. Use it as a quick reference when revising.

| Concept | Key Points to Remember |
|---------|----------------------|
| Subatomic particles | Atomic number Z = number of protons. Mass number A = protons + neutrons. In a neutral atom, electrons equal protons. |
| Isotopes | Same Z, different A. Same chemical properties (identical electron configuration). Different physical properties (different mass). |
| Relative atomic mass | Weighted average of all naturally occurring isotope masses: Aᵣ = Σ(mass × % abundance) / 100. |
| Mass spectrometer | Five stages: vaporisation → ionisation → acceleration → deflection → detection. Ions are separated by mass-to-charge ratio (m/z). Lighter ions and more highly charged ions deflect more. |
| Electron configuration | Aufbau principle: fill lowest energy orbitals first. Order: 1s, 2s, 2p, 3s, 3p, 4s, 3d, 4p. Hund's rule: electrons occupy separate orbitals before pairing, with parallel spins. Pauli exclusion principle: maximum two electrons per orbital, opposite spins. |
| Exceptions (Cr, Cu) | Chromium: [Ar] 4s¹ 3d⁵ (half-filled d sub-level stability). Copper: [Ar] 4s¹ 3d¹⁰ (fully-filled d sub-level stability). |
| Ions of transition metals | Remove electrons from 4s before 3d. Fe: [Ar] 4s² 3d⁶. Fe²⁺: [Ar] 3d⁶. Fe³⁺: [Ar] 3d⁵. |
| Emission spectra | Photon energy: ΔE = hν = hc/λ. Lyman series: n₁ = 1 (UV). Balmer series: n₁ = 2 (visible). Paschen series: n₁ = 3 (IR). |
| Convergence limit | Corresponds to n₂ = ∞. Energy at convergence (Lyman series) equals the first ionisation energy. |
| First ionisation energy | X(g) → X⁺(g) + e⁻. Increases across a period (increasing Z, constant shielding). Decreases down a group (increasing distance and shielding outweigh increasing Z). |
| IE anomalies | Be → B: 2p electron in B is easier to remove than 2s electron in Be. N → O: paired electron repulsion in O's 2p⁴ outweighs increased Z. |
| Successive IE | Always increase: IE₁ < IE₂ < IE₃ < … Large jump indicates breaking into an inner shell. Number of electrons before the jump = number of valence electrons = group number (for main-group elements). |
