Lesson 25: $K_c$ Calculations — ICE Tables & Exam Technique

Syllabus reference: 7.1, 17.1 — Equilibrium Calculations

Learning Objectives

By the end of this lesson, you will be able to:

  • Construct an ICE (Initial–Change–Equilibrium) table for any homogeneous equilibrium problem.
  • Calculate $K_c$ given initial amounts and one equilibrium concentration (or vice versa).
  • Solve quadratic equations arising from $K_c$ expressions where the change $x$ is not negligible.
  • Apply the "small $x$" approximation when $K_c$ is very small and justify when it is valid.
  • Handle $K_c$ problems where the reaction is approached from the product side ("reverse ICE").

1. Prior Knowledge Check

Question 1: In an ICE table, what do I, C, and E stand for? When are the "C" entries negative?

▸ Show
I = Initial concentrations (before any reaction), C = Change (the amount that reacts, using $x$), E = Equilibrium concentrations (I + C). C entries are negative for reactants (they are consumed) and positive for products (they are formed). The magnitude of each C entry is proportional to the stoichiometric coefficient: for $\ce{2A -> B}$, if A changes by $-2x$, B changes by $+x$.

Question 2: For $\ce{N2 + 3H2 <=> 2NH3}$, if $x$ mol of N₂ react, how much H₂ reacts and how much NH₃ forms?

▸ Show
From the stoichiometry (1:3:2), if $x$ mol of N₂ react, then $3x$ mol of H₂ react and $2x$ mol of NH₃ form. The C row would be: N₂ = $-x$, H₂ = $-3x$, NH₃ = $+2x$.

2. The ICE Table Method — Step by Step

The ICE table is the systematic method for solving equilibrium problems. It converts a verbal problem into algebra.

Figure 1: The ICE table for $\ce{A + B <=> 2C}$. The C row uses $x$ multiplied by stoichiometric coefficients. The E row feeds into the $K_c$ expression, producing an equation to solve.

2.1 The Systematic Approach

  1. Write the balanced equation and the $K_c$ expression.
  2. Convert all given amounts to concentrations (mol dm⁻³) by dividing by volume (dm³).
  3. Construct the ICE table. Define $x$ as the change in concentration of the species with coefficient 1, or the species whose equilibrium concentration is sought.
  4. Fill E row = I row + C row.
  5. Substitute E row into the $K_c$ expression and solve for $x$.
  6. Check your answer: concentrations must be positive. If you get two roots, discard the one that gives a negative concentration.
Figure 2: The five-step ICE method workflow. Always verify that all equilibrium concentrations are positive — discard any root that gives a negative value.

Check-in: For $\ce{A <=> 2B}$, a student writes C row: A = $-2x$, B = $+x$. What is wrong?

▸ Show
The changes must be proportional to the stoichiometric coefficients. For $\ce{A <=> 2B}$, if A changes by $-x$, B changes by $+2x$ (not $+x$). The student has the ratio reversed. The correct C row: A = $-x$, B = $+2x$. Think: for every 1 A that decomposes, 2 B are formed — so B's change must be twice A's.

3. Solving $K_c$ Problems — Three Common Scenarios

Scenario A: Given initial amounts and $K_c$, find equilibrium concentrations

Set up the ICE table, substitute E row into $K_c$, and solve. If $K_c$ is small and the initial concentration is large, you may be able to use the small-$x$ approximation: if $x \ll [\text{initial}]$, then $[\text{initial}] - x \approx [\text{initial}]$. Always check that $x$ is less than 5% of the initial concentration to justify the approximation.

Scenario B: Given initial amounts and one equilibrium concentration, find $K_c$

Use the known equilibrium concentration to find $x$ directly (no quadratic), then calculate all other equilibrium concentrations and substitute into $K_c$.

Scenario C: Reaction approached from products

The initial concentrations may be on the product side. In this case, the C row has negative entries for products and positive for reactants — the reaction runs "backwards" to reach equilibrium. The ICE method works identically; just be careful with signs.

Check-in: When is the small-$x$ approximation justified, and how do you verify it?

▸ Show
The approximation is justified when $K_c$ is very small (typically $K_c < 10^{-3}$) and initial concentrations are not extremely dilute. After solving, verify: if $(x / [\text{initial}]) \times 100\% < 5\%$, the approximation is valid. If it exceeds 5%, you must solve the full quadratic. Warning: Never apply the small-$x$ approximation if the initial concentration of the limiting species is less than about 100× $K_c$.

4. Worked Examples

Example 1 — Finding $K_c$ from One Equilibrium Concentration:

0.500 mol of $\ce{PCl5}$ is placed in a 2.00 dm³ sealed container and heated to 250°C. At equilibrium, 0.200 mol of $\ce{PCl5}$ remains: $\ce{PCl5(g) <=> PCl3(g) + Cl2(g)}$. Calculate $K_c$.

Solution using ICE:
PCl₅PCl₃Cl₂
I (mol dm⁻³)0.25000
C−x+x+x
E0.250−xxx
Given: at equilibrium, [PCl₅] = 0.200/2.00 = 0.100 mol dm⁻³. So 0.250 − x = 0.100 → $x = 0.150$.
Equilibrium: [PCl₅] = 0.100, [PCl₃] = 0.150, [Cl₂] = 0.150 mol dm⁻³.
$K_c = (0.150 \times 0.150) / 0.100 = \mathbf{0.225}$. Units: mol dm⁻³ (Δn = 2−1 = 1).

⚠️ Common wrong approach: Using moles directly without dividing by volume: $K_c = (0.300\times0.300)/0.200 = 0.450$. This gives the wrong $K_c$ because Δn ≠ 0.
Example 2 — Solving Quadratic for $K_c$:

For $\ce{H2(g) + I2(g) <=> 2HI(g)}$, 1.00 mol H₂ and 1.00 mol I₂ are placed in a 1.00 dm³ vessel at 440°C. $K_c = 49.5$. Find equilibrium concentrations.

Solution:
H₂I₂HI
I1.001.000
C−x−x+2x
E1.00−x1.00−x2x
$K_c = (2x)^2 / ((1.00-x)(1.00-x)) = 4x^2/(1.00-x)^2 = 49.5$.
Take square root: $2x/(1.00-x) = \sqrt{49.5} = 7.035$. → $2x = 7.035 - 7.035x$ → $9.035x = 7.035$ → $x = 0.7786$.
Equilibrium: [H₂] = [I₂] = $1.00 - 0.7786 = 0.221$ mol dm⁻³, [HI] = $2 \times 0.7786 = 1.557$ mol dm⁻³.
Check: $K_c = (1.557)^2/(0.221)^2 = 2.424/0.0488 = 49.6$ ✓ (rounding).

⚠️ Common wrong approach: Forgetting to square the coefficients in the C row. "If 2HI forms, change is +2x, NOT +x." Also: not taking the square root to simplify — you can solve $4x^2/(1-x)^2 = 49.5$ as a quadratic, but taking the square root of both sides first is much simpler when the numerator and denominator are both perfect squares.
Example 3 — Small-$x$ Approximation:

For $\ce{COCl2(g) <=> CO(g) + Cl2(g)}$, $K_c = 2.0 \times 10^{-6}$ at 400°C. Initially, [COCl₂] = 0.50 mol dm⁻³. Find equilibrium concentrations.

Solution: ICE: I = [0.50, 0, 0]; C = [−x, +x, +x]; E = [0.50−x, x, x].
$K_c = x^2/(0.50-x) = 2.0\times10^{-6}$. Since $K_c$ is tiny, $x$ is negligible compared to 0.50: $x^2/0.50 \approx 2.0\times10^{-6}$. → $x^2 = 1.0\times10^{-6}$ → $x = 1.0\times10^{-3}$.
Check: $x/0.50 = 0.001/0.50 = 0.2\% < 5\%$ → approximation valid.
Equilibrium: [COCl₂] ≈ 0.50, [CO] = [Cl₂] = $1.0 \times 10^{-3}$ mol dm⁻³.

⚠️ Common wrong approach: Applying the small-$x$ approximation when $K_c$ is moderate (e.g., $K_c = 0.1$). The approximation fails badly when $K_c$ is not very small. Always check! If the check fails, you must solve the full quadratic: $x^2 + K_c x - K_c[\text{initial}] = 0$.

⚠️ Common Mistakes to Avoid

  1. "I can use moles directly in $K_c$." You must convert to concentrations (mol dm⁻³) unless the volume is exactly 1.00 dm³ and Δn = 0. Using moles when Δn ≠ 0 and V ≠ 1.00 dm³ gives an incorrect $K_c$ value.
  2. "The C row always uses $+x$ for products and $-x$ for reactants." This is only true when starting from reactants. If you start from products (or a mixture), you must determine the direction of net reaction first (compare $Q$ to $K_c$) and assign C signs accordingly.
  3. "The small-$x$ approximation always works when $K_c$ is small." It also depends on the initial concentration. If the initial concentration is also very small (comparable to $K_c$), $x$ may not be negligible. The ratio $x/[\text{initial}]$ is the true test.
  4. "I can always avoid quadratics by taking square roots." Only when the $K_c$ expression is a perfect square — i.e., numerator and denominator are both squares of linear terms in $x$. Otherwise, you must solve a quadratic, which the IB expects you to do.
  5. "Both roots of a quadratic are valid equilibrium solutions." Only the root that gives all positive equilibrium concentrations is valid. The other root is mathematically correct but physically meaningless.

6. End-of-Lesson Practice

Question 1: 0.400 mol of $\ce{N2O4}$ is placed in a 4.00 dm³ container at 298 K: $\ce{N2O4(g) <=> 2NO2(g)}$. At equilibrium, [NO₂] = 0.060 mol dm⁻³. Calculate $K_c$.

Question 2: For $\ce{2SO3(g) <=> 2SO2(g) + O2(g)}$, $K_c = 0.040$ at 900 K. Initially, [SO₃] = 0.200 mol dm⁻³ with no SO₂ or O₂. Determine the equilibrium concentration of O₂.

Question 3: For $\ce{CO(g) + H2O(g) <=> CO2(g) + H2(g)}$, $K_c = 1.00$ at 1100 K. Initially, [CO] = [H₂O] = 0.500 mol dm⁻³ with no products. Find all equilibrium concentrations.

Question 4: 0.100 mol of HI is placed in a 1.00 dm³ vessel at 700 K: $\ce{2HI(g) <=> H2(g) + I2(g)}$. $K_c = 0.0183$. Find the equilibrium concentration of HI.

Question 5: For $\ce{2NO2(g) <=> N2O4(g)}$, $K_c = 216$ at 298 K. Initially, [NO₂] = 0.200 mol dm⁻³, [N₂O₄] = 0. Calculate the equilibrium concentration of N₂O₄. (This will require solving a quadratic.)

Question 6 (IB multi-step): 0.600 mol of $\ce{PCl5}$ is heated to 500 K in a 2.00 dm³ vessel. At equilibrium, the mixture contains 0.200 mol of Cl₂: $\ce{PCl5(g) <=> PCl3(g) + Cl2(g)}$.
(a) Set up an ICE table and find $x$.
(b) Calculate all equilibrium concentrations.
(c) Calculate $K_c$ with correct units.
(d) If an additional 0.100 mol of Cl₂ is injected at constant volume, predict the direction of net reaction using $Q$.

Answer 1: [N₂O₄]₀ = 0.400/4.00 = 0.100 mol dm⁻³. ICE: I = [0.100, 0]; C = [−x, +2x]; E = [0.100−x, 2x]. Given 2x = 0.060 → x = 0.030. [N₂O₄]eq = 0.070. $K_c = (0.060)^2/0.070 = 0.0514$ mol dm⁻³.

Answer 2: ICE: I = [0.200, 0, 0]; C = [−2x, +2x, +x]; E = [0.200−2x, 2x, x]. $K_c = (2x)^2(x)/(0.200-2x)^2 = 4x^3/(0.200-2x)^2 = 0.040$. This is a cubic — but with small $K_c$, try approximation: $4x^3/(0.200)^2 \approx 0.040$ → $4x^3 = 0.0016$ → $x^3 = 0.0004$ → $x = 0.0737$. Check: $2x = 0.147$, compared to 0.200 — too large (74%), so approximation fails. Solve properly: this is exam-level difficult. But for small $K_c$, the assumption that $2x \ll 0.200$ often holds; here it does not, indicating a significant decomposition. The exact solution requires an iterative or cubic approach — this problem illustrates when NOT to use the small-x approximation.

Answer 3: ICE: I = [0.500, 0.500, 0, 0]; C = [−x, −x, +x, +x]; E = [0.500−x, 0.500−x, x, x]. $K_c = x^2/(0.500-x)^2 = 1.00$. Square root: $x/(0.500-x) = 1$ → $x = 0.500-x$ → $2x = 0.500$ → $x = 0.250$. All equilibrium concentrations = 0.250 mol dm⁻³.

Answer 4: ICE: I = [0.100, 0, 0]; C = [−2x, +x, +x]; E = [0.100−2x, x, x]. $K_c = x^2/(0.100-2x)^2 = 0.0183$. Square root: $x/(0.100-2x) = \sqrt{0.0183} = 0.1353$. $x = 0.01353 - 0.2706x$ → $1.2706x = 0.01353$ → $x = 0.01065$. [HI]eq = $0.100 - 2(0.01065) = \mathbf{0.0787}$ mol dm⁻³.

Answer 5: ICE: I = [0.200, 0]; C = [−2x, +x]; E = [0.200−2x, x]. $K_c = x/(0.200-2x)^2 = 216$. $x = 216(0.200-2x)^2 = 216(0.04 - 0.8x + 4x^2)$. $x = 8.64 - 172.8x + 864x^2$. $864x^2 - 173.8x + 8.64 = 0$. Quadratic formula gives $x \approx 0.094$ (the root giving [NO₂] positive). [N₂O₄]eq = 0.094 mol dm⁻³.

Answer 6: (a) [PCl₅]₀ = 0.300. ICE: I=[0.300,0,0]; C=[−x,+x,+x]; E=[0.300−x, x, x]. Given [Cl₂]eq = 0.200/2.00 = 0.100 → $x = 0.100$. (b) [PCl₅]eq = 0.200, [PCl₃]eq = 0.100, [Cl₂]eq = 0.100. (c) $K_c = (0.100\times0.100)/0.200 = 0.0500$ mol dm⁻³. (d) New [Cl₂] = 0.100 + 0.050 = 0.150. $Q = (0.100\times0.150)/0.200 = 0.075$. $Q > K_c$ → net reaction shifts left.

7. Key Terms

TermDefinition
ICE tableA systematic grid (Initial, Change, Equilibrium) for organising equilibrium concentration calculations
$x$ (change variable)The unknown change in concentration of one species; scaled by stoichiometric coefficients for others
Small-$x$ approximationSimplifying $[\text{initial}] - x \approx [\text{initial}]$ when $x$ is negligible; valid only if $x < 5\%$ of initial
Quadratic formula$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$; used when $K_c$ yields a quadratic equation in $x$
Reverse ICEICE table where initial concentrations are of products; C row has opposite signs

8. Exam Tips

Paper 2 (Structured): ICE table questions are worth 4–6 marks. Always: (1) write the balanced equation, (2) convert to concentrations, (3) draw the full ICE table with clear column headings, (4) show the $K_c$ substitution step, (5) solve for $x$ showing work, (6) state all equilibrium concentrations with units, (7) give $K_c$ with correct units. Partial marks are awarded for correct ICE structure and $K_c$ expression even if algebra is incorrect.
Quadratic Equations: The IB expects you to solve quadratics. Organise them carefully: bring all terms to one side, identify $a$, $b$, $c$, and apply the formula. The negative root is almost always the one to discard. Write: "$x = \text{[value]}$ or $x = \text{[value]}$. The second value gives negative concentrations and is rejected."
Checking Your Answer: Always substitute your final equilibrium concentrations back into the $K_c$ expression to verify you get approximately the given $K_c$. A discrepancy of more than 5% suggests an algebra error. This 30-second check can save you 3–4 marks on an exam.
