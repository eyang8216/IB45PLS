Lesson 33: Balancing Redox Equations — The Half-Equation Method

Syllabus reference: 9.1–9.2 (HL) — Half-Equations, Redox Titrations

Learning Objectives

By the end of this lesson, you will be able to:

  • Split a redox reaction into its oxidation and reduction half-equations.
  • Balance half-equations for atoms, oxygen (using $\ce{H2O}$), hydrogen (using $\ce{H+}$), and charge (using electrons).
  • Combine half-equations to produce the balanced overall redox equation, cancelling electrons.
  • Adapt half-equations for reactions in basic solution by adding $\ce{OH-}$ to neutralise $\ce{H+}$.
  • Apply stoichiometric ratios from balanced redox equations in titration calculations.

1. Prior Knowledge Check

Question 1: In the reaction $\ce{Zn + Cu^{2+} -> Zn^{2+} + Cu}$, write the oxidation half-equation and the reduction half-equation.

▸ Show
Oxidation: $\ce{Zn -> Zn^{2+} + 2e-}$ (Zn loses 2 electrons). Reduction: $\ce{Cu^{2+} + 2e- -> Cu}$ (Cu²⁺ gains 2 electrons). Combined: $\ce{Zn + Cu^{2+} -> Zn^{2+} + Cu}$. Electrons cancel.

Question 2: What species provide $\ce{H+}$ and $\ce{H2O}$ when balancing half-equations in acidic solution?

▸ Show
In acidic solution: use $\ce{H2O}$ to balance oxygen atoms, then use $\ce{H+}$ to balance hydrogen atoms. In basic solution: balance as if in acid first, then add $\ce{OH-}$ to both sides to neutralise all $\ce{H+}$ (converting them to $\ce{H2O}$).

2. The Half-Equation Method — Step-by-Step

Many redox reactions — especially those involving oxyanions like $\ce{MnO4-}$, $\ce{Cr2O7^{2-}}$, $\ce{NO3-}$ — are impossible to balance by inspection. The half-equation method provides a systematic approach.

Step-by-Step for Acidic Solution

  1. Write the skeleton half-equations — one for oxidation, one for reduction. Include only the species that change oxidation number.
  2. Balance all atoms EXCEPT H and O.
  3. Balance O atoms by adding $\ce{H2O}$ molecules.
  4. Balance H atoms by adding $\ce{H+}$ ions.
  5. Balance charge by adding electrons ($e^-$) to the more positive side.
  6. Multiply each half-equation so the number of electrons is the same in both.
  7. Add the half-equations, cancelling electrons and any species appearing on both sides.
Figure 1: The seven-step algorithm for balancing redox equations. Follow in strict order — never skip a step.

Check-in: Balance $\ce{MnO4- -> Mn^{2+}}$ (reduction half-equation in acid).

▸ Show
1. Mn already balanced (1 each side).
2. Balance O: 4 O on left → add 4 H₂O to right: $\ce{MnO4- -> Mn^{2+} + 4H2O}$.
3. Balance H: 8 H on right → add 8 H⁺ to left: $\ce{MnO4- + 8H+ -> Mn^{2+} + 4H2O}$.
4. Balance charge: left (−1 + 8 = +7), right (+2). Add 5 e⁻ to left: $\ce{MnO4- + 8H+ + 5e- -> Mn^{2+} + 4H2O}$. Done.

3. Worked Full Example — Acidic Conditions

Balance: $\ce{Fe^{2+} + MnO4- -> Fe^{3+} + Mn^{2+}}$ (acidic)

Oxidation half: $\ce{Fe^{2+} -> Fe^{3+} + e-}$ (Fe and charge already balanced — 1 e⁻ needed).

Reduction half: $\ce{MnO4- + 8H+ + 5e- -> Mn^{2+} + 4H2O}$ (as derived above).

Equalise electrons: Multiply oxidation by 5: $\ce{5Fe^{2+} -> 5Fe^{3+} + 5e-}$.

Add: $\ce{5Fe^{2+} + MnO4- + 8H+ + 5e- -> 5Fe^{3+} + Mn^{2+} + 4H2O + 5e-}$.

Cancel 5e⁻: $\ce{5Fe^{2+} + MnO4- + 8H+ -> 5Fe^{3+} + Mn^{2+} + 4H2O}$.

4. Balancing in Basic Solution

For reactions in basic solution, first balance as if in acid (using $\ce{H+}$ and $\ce{H2O}$), then add $\ce{OH-}$ to both sides to convert every $\ce{H+}$ to $\ce{H2O}$. If $\ce{OH-}$ appears, cancel any $\ce{H2O}$ that appears on both sides.

Check-in: Balance $\ce{MnO4- + SO3^{2-} -> MnO2 + SO4^{2-}}$ in basic solution.

▸ Show
Reduction: $\ce{MnO4- + 2H2O + 3e- -> MnO2 + 4OH-}$. Oxidation: $\ce{SO3^{2-} + 2OH- -> SO4^{2-} + H2O + 2e-}$. Equalise: multiply reduction by 2, oxidation by 3. Add: $\ce{2MnO4- + 3SO3^{2-} + H2O -> 2MnO2 + 3SO4^{2-} + 2OH-}$.
Figure 2: Converting an acidic half-equation to basic conditions. Add OH⁻ to neutralise all H⁺, then cancel water molecules on both sides.

5. Worked Examples

Example 1 — Dichromate with Iron(II) in Acid:

Balance: $\ce{Cr2O7^{2-} + Fe^{2+} -> Cr^{3+} + Fe^{3+}}$ (acidic)

Reduction: $\ce{Cr2O7^{2-} -> 2Cr^{3+}}$. Balance O: +7H₂O right. Balance H: +14H⁺ left. Balance charge: left −2 + 14 = +12, right +6 → add 6e⁻: $\ce{Cr2O7^{2-} + 14H+ + 6e- -> 2Cr^{3+} + 7H2O}$.
Oxidation: $\ce{Fe^{2+} -> Fe^{3+} + e-}$. Multiply by 6.
Combined: $\ce{Cr2O7^{2-} + 6Fe^{2+} + 14H+ -> 2Cr^{3+} + 6Fe^{3+} + 7H2O}$.

⚠️ Common wrong approach: Forgetting to balance Cr atoms first (Cr₂O₇²⁻ → 2Cr³⁺). Also, adding 6 e⁻ to the reduction is correct — Cr goes from +6 to +3 (×2 atoms = 6 e⁻ total).
Example 2 — Disproportionation of Chlorine in Base:

Balance: $\ce{Cl2 -> Cl- + ClO-}$ in basic solution.

Reduction: $\ce{Cl2 + 2e- -> 2Cl-}$.
Oxidation: $\ce{Cl2 -> 2ClO-}$. Balance O: +2H₂O left: $\ce{Cl2 + 2H2O -> 2ClO-}$. Balance H: +4H⁺ right: $\ce{Cl2 + 2H2O -> 2ClO- + 4H+}$. Balance charge: right (−2 + 4 = +2) → add 2e⁻: $\ce{Cl2 + 2H2O -> 2ClO- + 4H+ + 2e-}$.
In base: Add 4OH⁻ both sides → $\ce{Cl2 + 2H2O + 4OH- -> 2ClO- + 4H2O + 2e-}$. Cancel 2H₂O: $\ce{Cl2 + 4OH- -> 2ClO- + 2H2O + 2e-}$.
Combine with reduction: $\ce{2Cl2 + 4OH- -> 2Cl- + 2ClO- + 2H2O}$. Divide by 2: $\ce{Cl2 + 2OH- -> Cl- + ClO- + H2O}$.

⚠️ Common wrong approach: Not realising this is disproportionation — Cl goes from 0 to +1 (oxidised) and 0 to −1 (reduced). Each Cl₂ molecule must appear in both half-equations.
Example 3 — Redox Titration Calculation:

25.0 cm³ of 0.0200 mol dm⁻³ $\ce{KMnO4}$ reacts completely with 20.0 cm³ of $\ce{FeSO4}$ solution in acid. Calculate the concentration of $\ce{FeSO4}$.

$\ce{MnO4- + 5Fe^{2+} + 8H+ -> Mn^{2+} + 5Fe^{3+} + 4H2O}$

Step 1: $n(\ce{MnO4-}) = 0.0200 \times 0.0250 = 5.00\times10^{-4}$ mol.
Step 2: Mole ratio 1:5. $n(\ce{Fe^{2+}}) = 5 \times 5.00\times10^{-4} = 2.50\times10^{-3}$ mol.
Step 3: $[\ce{Fe^{2+}}] = 2.50\times10^{-3} / 0.0200 = \mathbf{0.125\text{ mol dm}^{-3}}$.

⚠️ Common wrong approach: Using a 1:1 ratio instead of 1:5. The stoichiometry comes from the balanced equation — always derive the ratio from the number of electrons transferred.

⚠️ Common Mistakes to Avoid

  1. Skipping the charge balance step. The most common error is balancing atoms but forgetting electrons. A half-equation is only correct when both atoms AND charge are balanced.
  2. Using the wrong mole ratio in redox titrations. The ratio comes from the electrons transferred, not from the coefficients of the unbalanced equation. Always write the balanced equation first.
  3. Forgetting to convert to basic conditions when required. If the reaction says "in basic solution", you must add OH⁻ after balancing as if in acid. Simply adding OH⁻ at the end without the acid→base conversion gives the wrong equation.
  4. Adding H⁺ and OH⁻ to the same side. When converting to basic conditions, add OH⁻ to BOTH sides. Then H⁺ + OH⁻ → H₂O on whichever side H⁺ was. Never have both H⁺ and OH⁻ on the same side in the final equation.

Question 2: Balance $\ce{Cr2O7^{2-} + I- -> Cr^{3+} + I2}$ in acidic solution.

Question 3: Balance $\ce{MnO4- + C2O4^{2-} -> MnO2 + CO3^{2-}}$ in basic solution.

Question 4: Balance the disproportionation: $\ce{H2O2 -> H2O + O2}$ (acidic). Show both half-equations.

Question 5 (Redox titration): 25.0 cm³ of acidified $\ce{K2Cr2O7}$ (0.0167 mol dm⁻³) oxidises 30.0 cm³ of $\ce{Fe^{2+}}$ solution. Calculate $[\ce{Fe^{2+}}]$. The half-equation for Cr₂O₇²⁻ → Cr³⁺ requires 6 e⁻.

Answer 1: $\ce{4Zn + NO3- + 10H+ -> 4Zn^{2+} + NH4+ + 3H2O}$.
Reduction: $\ce{NO3- + 10H+ + 8e- -> NH4+ + 3H2O}$. Oxidation: $\ce{Zn -> Zn^{2+} + 2e-}$ (×4).

Answer 2: $\ce{Cr2O7^{2-} + 6I- + 14H+ -> 2Cr^{3+} + 3I2 + 7H2O}$.
Reduction: $\ce{Cr2O7^{2-} + 14H+ + 6e- -> 2Cr^{3+} + 7H2O}$. Oxidation: $\ce{2I- -> I2 + 2e-}$ (×3).

Answer 3: $\ce{2MnO4- + 3C2O4^{2-} + 2H2O -> 2MnO2 + 6CO3^{2-} + 4H+}$ then convert to base: $\ce{2MnO4- + 3C2O4^{2-} + 4OH- -> 2MnO2 + 6CO3^{2-} + 2H2O}$.

Answer 4: Oxidation: $\ce{H2O2 -> O2 + 2H+ + 2e-}$. Reduction: $\ce{H2O2 + 2H+ + 2e- -> 2H2O}$. Combining: $\ce{2H2O2 -> 2H2O + O2}$.

Answer 5: $n(\ce{Cr2O7^{2-}}) = 0.0167 \times 0.0250 = 4.175\times10^{-4}$ mol. $\ce{Cr2O7^{2-} + 6Fe^{2+} + 14H+ -> 2Cr^{3+} + 6Fe^{3+} + 7H2O}$. Ratio 1:6. $n(\ce{Fe^{2+}}) = 6 \times 4.175\times10^{-4} = 2.505\times10^{-3}$ mol. $[\ce{Fe^{2+}}] = 2.505\times10^{-3}/0.0300 = \mathbf{0.0835\text{ mol dm}^{-3}}$.

7. Key Terms

TermDefinition
Half-equationAn equation showing either oxidation or reduction alone, including electrons
Oxidation halfThe half-equation where electrons appear on the product side (electrons lost)
Reduction halfThe half-equation where electrons appear on the reactant side (electrons gained)
Acidic balancingUsing $\ce{H2O}$ and $\ce{H+}$ to balance O and H; standard for most redox reactions
Basic balancingFirst balance as acid, then add $\ce{OH-}$ to both sides to neutralise $\ce{H+}$
Redox titrationA titration based on a redox reaction; the equivalence point is detected by colour change (often self-indicating, e.g., $\ce{MnO4-}$ purple → colourless)
DisproportionationA redox reaction where the same species is both oxidised and reduced

8. Exam Tips

Paper 2 (Structured): Balancing redox equations is typically 3–4 marks. You earn marks for: (1) correct half-equations with proper electron counts, (2) correct H⁺/H₂O balance, (3) multiplying and combining correctly, (4) the final balanced equation. Even if the final equation is wrong, half-equation marks are awarded independently.
Redox Titrations: These appear in Paper 2 Section A. The mole ratio always comes from the balanced redox equation. Common examined pairs: $\ce{MnO4-/Fe^{2+}}$ (1:5), $\ce{Cr2O7^{2-}/Fe^{2+}}$ (1:6), $\ce{I2/S2O3^{2-}}$ (1:2). Memorise these ratios — they save time.
Self-indicating titrations: $\ce{KMnO4}$ titrations need no indicator — the intense purple colour of $\ce{MnO4-}$ disappears as it's reduced to colourless $\ce{Mn^{2+}}$. The end point is the first permanent pink colour. Iodine titrations use starch indicator (deep blue-black complex with $\ce{I2}$).
