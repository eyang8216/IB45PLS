Lesson 32: Oxidation Numbers & Identifying Redox Reactions

Syllabus reference: 9.1 — Oxidation and Reduction, Oxidation States

Learning Objectives

By the end of this lesson, you will be able to:

  • Define oxidation and reduction in terms of electron transfer and oxidation number changes.
  • Apply the rules for assigning oxidation numbers to atoms in any compound or ion.
  • Identify which species is oxidised and which is reduced in a redox reaction.
  • Distinguish redox reactions from non-redox reactions (e.g., precipitation, acid–base, ligand exchange).
  • Determine the oxidising agent and reducing agent in a reaction.

1. Prior Knowledge Check

Question 1: What does OIL RIG stand for?

▸ Show
Oxidation Is Loss (of electrons). Reduction Is Gain (of electrons). This is the electron-transfer definition. The species that loses electrons is oxidised; the species that gains electrons is reduced.

Question 2: What is the charge on (a) an oxide ion, (b) a chloride ion, (c) a sodium ion?

▸ Show
(a) $\ce{O^{2-}}$, (b) $\ce{Cl-}$, (c) $\ce{Na+}$. These are the common monatomic ions — their oxidation numbers equal their charges.

2. The Oxidation Number — A Bookkeeping Tool

The oxidation number (or oxidation state) is the apparent charge an atom would have if all bonds were completely ionic — with the more electronegative atom taking all the bonding electrons. It is a formalism, not a real charge, but it is indispensable for tracking electron transfer.

2.1 The Nine Rules for Assigning Oxidation Numbers

Apply these rules in order — when a conflict arises, the rule that appears earlier takes priority:

#RuleExample
1The oxidation number of an atom in a free element is 0.$\ce{O2}$: O = 0; $\ce{Na(s)}$: Na = 0; $\ce{P4}$: P = 0
2For a monatomic ion, the oxidation number equals the charge of the ion.$\ce{Cl-}$: −1; $\ce{Fe^{3+}}$: +3; $\ce{S^{2-}}$: −2
3Fluorine in compounds is always −1.$\ce{HF}$, $\ce{OF2}$, $\ce{NaF}$: F = −1
4Group 1 metals (Li, Na, K, etc.) are always +1. Group 2 (Mg, Ca, etc.) are always +2.$\ce{NaCl}$: Na = +1; $\ce{CaO}$: Ca = +2
5Hydrogen is +1 when bonded to non-metals, −1 when bonded to metals (hydrides).$\ce{H2O}$: H = +1; $\ce{NaH}$: H = −1
6Oxygen is −2 in most compounds. Exceptions: peroxides (−1), superoxides (−½), $\ce{OF2}$ (+2).$\ce{H2O}$: O = −2; $\ce{H2O2}$: O = −1
7Group 17 (halogens) are usually −1, except when bonded to O or a more electronegative halogen.$\ce{NaCl}$: Cl = −1; $\ce{ClO-}$: Cl = +1
8The sum of oxidation numbers in a neutral compound is 0.$\ce{H2SO4}$: 2(+1) + S + 4(−2) = 0 → S = +6
9The sum of oxidation numbers in a polyatomic ion equals the ion's charge.$\ce{MnO4-}$: Mn + 4(−2) = −1 → Mn = +7
Figure 1: Oxidation number changes. An increase means oxidation (loss of electrons). A decrease means reduction (gain of electrons).

Check-in: Assign oxidation numbers to all atoms in $\ce{KMnO4}$.

▸ Show
K (Group 1): +1. O: −2 each (4 oxygen atoms). Sum must be 0: +1 + Mn + 4(−2) = 0 → Mn − 7 = 0 → Mn = +7. So $\ce{KMnO4}$: K = +1, Mn = +7, O = −2.

Check-in: What is the oxidation number of Cr in $\ce{Cr2O7^{2-}}$?

▸ Show
O: −2 each (7 O → −14 total). Sum must equal ion charge: 2Cr + 7(−2) = −2 → 2Cr − 14 = −2 → 2Cr = +12 → Cr = +6.

3. Identifying Redox Reactions

A reaction is a redox reaction if any atom changes its oxidation number between reactants and products. If no oxidation numbers change, the reaction is not redox — it may be precipitation, acid–base, or ligand exchange.

Check-in: Is $\ce{NaCl(aq) + AgNO3(aq) -> AgCl(s) + NaNO3(aq)}$ a redox reaction?

▸ Show
No. Check oxidation numbers: Na = +1 throughout, Cl = −1 throughout, Ag = +1 throughout, N in NO₃⁻ = +5 throughout, O = −2 throughout. No oxidation number changes → not redox — it's a precipitation reaction.

4. Oxidising and Reducing Agents

  • The oxidising agent (oxidant) is the species that causes oxidation of another species — it is itself reduced (gains electrons).
  • The reducing agent (reductant) is the species that causes reduction of another species — it is itself oxidised (loses electrons).
Memory: "The agent does the opposite of what happens to itself."
Figure 2: The reducing agent is oxidised; the oxidising agent is reduced. The names describe what they do to the OTHER species.

5. Worked Examples

Example 1 — Oxidation Numbers in an Organic Compound:

Find the oxidation number of each carbon atom in $\ce{CH3CH2OH}$ (ethanol).

Method: H = +1, O = −2 always. For each carbon, solve using the atoms directly attached.
C1 (CH₃−): bonded to 3 H (+1 each) and 1 C (0). Sum: C1 + 3(+1) + 0 = 0 → C1 = −3.
C2 (−CH₂OH): bonded to 2 H (+1 each), 1 O (−2), 1 C (0). Sum: C2 + 2(+1) + (−2) + 0 = 0 → C2 = −1.

⚠️ Common wrong approach: Treating all carbons as having the same oxidation number. In organic molecules, different carbon atoms can have different oxidation states — you must treat them individually.
Example 2 — Identifying the Redox Process:

$\ce{2Fe^{3+}(aq) + 2I-(aq) -> 2Fe^{2+}(aq) + I2(aq)}$
Identify: (a) what is oxidised, (b) what is reduced, (c) the oxidising agent, (d) the reducing agent.

Oxidation numbers: Fe: +3 → +2 (decrease of 1 — reduction). I: −1 → 0 (increase of 1 — oxidation).
(a) Oxidised: I⁻ (loses electrons, ON increases from −1 to 0).
(b) Reduced: Fe³⁺ (gains electrons, ON decreases from +3 to +2).
(c) Oxidising agent: Fe³⁺ (it causes oxidation of I⁻).
(d) Reducing agent: I⁻ (it causes reduction of Fe³⁺).

⚠️ Common wrong approach: Saying Fe³⁺ is the reducing agent because it gets reduced. The agent is named for what it DOES to the other species — the oxidising agent oxidises the other (and is itself reduced).
Example 3 — Is It Redox?

Classify each as redox or not:
(a) $\ce{CaCO3 ->[\Delta] CaO + CO2}$
(b) $\ce{2Na + Cl2 -> 2NaCl}$
(c) $\ce{HCl + NaOH -> NaCl + H2O}$

(a) Ca: +2 → +2; C: +4 → +4; O: −2 → −2. No changes → not redox (thermal decomposition).
(b) Na: 0 → +1 (oxidised); Cl: 0 → −1 (reduced). Changes → redox.
(c) H: +1 → +1; Cl: −1 → −1; Na: +1 → +1; O: −2 → −2. No changes → not redox (acid–base neutralisation).

⚠️ Common wrong approach: Assuming any reaction that forms a new substance is redox. Many reactions — precipitation, acid–base, ligand exchange, thermal decomposition — are NOT redox.

⚠️ Common Mistakes to Avoid

  1. Confusing the agent with the process. The oxidising agent is reduced. The reducing agent is oxidised. Think: the agent "does" what its name says to the other species.
  2. Forgetting that oxidation numbers are per atom, not per formula unit. In $\ce{Cr2O7^{2-}}$, each Cr is +6 — not +12. Divide by the subscript when assigning to individual atoms.
  3. Assuming oxygen is always −2. In peroxides ($\ce{H2O2}$, $\ce{Na2O2}$) it's −1. In superoxides ($\ce{KO2}$) it's −½. In $\ce{OF2}$ it's +2 (fluorine is more electronegative).
  4. Treating all reactions as redox. Precipitation ($\ce{Ag+ + Cl- -> AgCl}$), acid–base neutralisation, and ligand exchange reactions involve no change in oxidation numbers.

Question 2: In the reaction $\ce{Zn + CuSO4 -> ZnSO4 + Cu}$, identify what is oxidised, what is reduced, the oxidising agent, and the reducing agent.

Question 3: Which of these are redox reactions? (a) $\ce{2H2 + O2 -> 2H2O}$, (b) $\ce{CaO + H2O -> Ca(OH)2}$, (c) $\ce{Cl2 + 2KBr -> 2KCl + Br2}$.

Question 4: Determine the oxidation number of: (a) N in $\ce{NH4+}$, (b) S in $\ce{S2O3^{2-}}$, (c) Fe in $\ce{Fe3O4}$.

Question 5 (Paper 1 style): In which compound does chlorine have an oxidation number of +5?
A. $\ce{HCl}$   B. $\ce{HClO}$   C. $\ce{HClO2}$   D. $\ce{HClO3}$

Question 6: $\ce{5Fe^{2+} + MnO4- + 8H+ -> 5Fe^{3+} + Mn^{2+} + 4H2O}$. (a) Which species is the oxidising agent? (b) By how many units does the oxidation number of Mn change?

Answer 1: (a) SO₂: O = −2 each, S + 2(−2) = 0 → S = +4. (b) H₂SO₄: H = +1, O = −2, 2(+1) + S + 4(−2) = 0 → S = +6. (c) MnO₂: O = −2, Mn + 2(−2) = 0 → Mn = +4. (d) Cr₂O₇²⁻: O = −2, 2Cr + 7(−2) = −2 → 2Cr = +12 → Cr = +6. (e) VO₂⁺: O = −2, V + 2(−2) = +1 → V = +5.

Answer 2: Zn: 0 → +2 (oxidised — loses 2 e⁻). Cu: +2 → 0 (reduced — gains 2 e⁻). Reducing agent: Zn (causes Cu²⁺ to be reduced). Oxidising agent: Cu²⁺ (causes Zn to be oxidised).

Answer 3: (a) Redox — H: 0→+1 (oxidised), O: 0→−2 (reduced). (b) NOT redox — Ca: +2→+2, O: −2→−2, H: +1→+1. (c) Redox — Cl: 0→−1 (reduced), Br: −1→0 (oxidised).

Answer 4: (a) NH₄⁺: H = +1, N + 4(+1) = +1 → N = −3. (b) S₂O₃²⁻: O = −2, 2S + 3(−2) = −2 → 2S = +4 → S = +2. (c) Fe₃O₄: O = −2, 3Fe + 4(−2) = 0 → 3Fe = +8 → average Fe = +8/3. Actually Fe₃O₄ is FeO·Fe₂O₃: two Fe³⁺ (+3 each) and one Fe²⁺ (+2).

Answer 5: D. HClO₃: H = +1, O = −2 each. +1 + Cl + 3(−2) = 0 → Cl = +5. (A: −1, B: +1, C: +3).

Answer 6: (a) MnO₄⁻ is the oxidising agent (Mn is reduced from +7 to +2, causing Fe²⁺ to be oxidised). (b) Mn goes from +7 to +2 — a decrease of 5 units, meaning each Mn gains 5 electrons.

7. Key Terms

TermDefinition
Oxidation numberThe apparent charge on an atom if all bonds were treated as completely ionic
OxidationLoss of electrons; an increase in oxidation number
ReductionGain of electrons; a decrease in oxidation number
Redox reactionA reaction in which at least one atom changes its oxidation number
Oxidising agentThe species that gains electrons (is reduced) and causes oxidation of another species
Reducing agentThe species that loses electrons (is oxidised) and causes reduction of another species
DisproportionationA redox reaction where the same element is both oxidised and reduced
PeroxideA compound containing O–O bond where oxygen has oxidation number −1

8. Exam Tips

Paper 1 (Multiple Choice): Questions often test exceptions to the rules — especially peroxides (O = −1), metal hydrides (H = −1), and compounds like $\ce{OF2}$ (O = +2). The most common trap is asking for the oxidation number in a polyatomic ion and giving answer choices that correspond to the sum rather than per atom (e.g., Cr = +6, not +12, in Cr₂O₇²⁻).
Paper 2 (Structured): When asked to identify the oxidising/reducing agent, always name the species, not the element. For example, "MnO₄⁻ is the oxidising agent", not "Mn is the oxidising agent". The distinction between "oxidised" (the process) and "oxidising agent" (the species) is a mark that students frequently lose.
Disproportionation: This is a favourite IB concept. Look for reactions where the same element appears in three different oxidation states — one intermediate state splits into a higher and lower state. Classic examples: $\ce{2H2O2 -> 2H2O + O2}$ and $\ce{Cl2 + 2NaOH -> NaCl + NaClO + H2O}$.
